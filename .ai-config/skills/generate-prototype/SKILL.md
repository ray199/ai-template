# 技能：生成原型

## 技能描述
该技能用于根据用户需求生成前端原型，包括页面结构、交互逻辑和视觉设计。

## 触发指令
- `/prototype` - 生成前端原型
- `/ui` - 生成UI设计
- `/mock` - 生成模拟数据

## 输入格式
```
/prototype

页面需求：
- 登录页面：包含用户名、密码输入框和登录按钮
- 首页：包含导航栏、内容区域和页脚
- 详情页：展示详细信息

技术栈（可选，不填则自动推荐）：
- 前端：React + Tailwind CSS
- 交互：React Hooks
```

## 执行流程

```
1. 解析需求 → 识别产品类型（企业系统 / SaaS / 电商 / 大屏 / 移动 / 内容）
   ↓
2. 【设计风格选择】调用 ui-ux-pro-max
   ├─ 根据产品类型推荐 3 个风格方案
   │  方案 A — [风格名]：色板 [hex/hex/hex]，字体 [标题+正文]，适合 [场景]
   │  方案 B — [风格名]：...
   │  方案 C — [风格名]：...
   │  方案 D — 自定义：输入偏好
   └─ 等待用户选择（A / B / C / D）
   ↓
3. 将选定 Token 注入 HTML :root 变量块
   ↓
4. 生成 HTML 原型（所有样式引用 CSS 变量）
   ↓
5. UX 自检（ui-ux-pro-max 优先级 1-3：无障碍 / 触控目标 / 响应式）
   ↓
6. 输出原型文件 + 所选风格说明
```

## 输出格式
- 完整 HTML 文件（CSS 变量 Token 体系，可一键切换风格）
- 包含模拟数据
- 基本交互逻辑（模态框、标签页、表单提交反馈）
- 响应式设计，移动端触控目标 ≥ 44px
- 文件末尾注释说明所选风格方案

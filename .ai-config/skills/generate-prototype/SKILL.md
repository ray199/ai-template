# 技能：生成原型

## 技能描述
该技能用于根据用户需求生成前端原型，包括页面结构、交互逻辑和视觉设计。

## 触发指令
- `/prototype` - 生成前端原型
- `/ui` - 生成UI设计
- `/mock` - 生成模拟数据

## 输入格式
```
/prototype

页面需求：
- 登录页面：包含用户名、密码输入框和登录按钮
- 首页：包含导航栏、内容区域和页脚
- 详情页：展示详细信息

技术栈：
- 前端：React + Tailwind CSS
- 交互：React Hooks
```

## 输出格式
- 生成HTML/CSS/JavaScript原型文件
- 包含模拟数据
- 实现基本交互逻辑
- 响应式设计

## 实现逻辑
1. 分析页面需求
2. 生成HTML结构
3. 添加Tailwind CSS样式
4. 实现基本交互
5. 生成模拟数据
6. 提供完整的原型文件

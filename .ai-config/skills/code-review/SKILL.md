# 技能：代码审查

## 技能描述
该技能用于对代码进行全面审查，包括代码质量、安全性、性能和最佳实践等方面。

## 触发指令
- `/review` - 审查代码
- `/audit` - 安全审计
- `/optimize` - 性能优化建议

## 输入格式
```
/review

代码文件：
- src/components/UserProfile.jsx
- src/services/apiClient.js

审查重点：
- 代码质量
- 安全性
- 性能
- 最佳实践
```

## 输出格式
```
# 代码审查报告

## 总体评价
- 代码质量：良好
- 安全性：中等
- 性能：良好
- 最佳实践：基本遵循

## 具体问题

### src/components/UserProfile.jsx
1. **性能问题**：组件渲染过于频繁
   - 建议：使用React.memo或useMemo优化
   
2. **代码风格**：变量命名不规范
   - 建议：使用驼峰命名法

### src/services/apiClient.js
1. **安全问题**：未处理API错误
   - 建议：添加错误处理逻辑
   
2. **性能问题**：重复请求
   - 建议：实现请求缓存

## 改进建议
1. 优化组件性能
2. 增强错误处理
3. 规范代码风格
4. 实现请求缓存
```

## 实现逻辑
1. 分析代码文件
2. 检查代码质量
3. 评估安全性
4. 分析性能问题
5. 检查最佳实践
6. 生成详细的审查报告

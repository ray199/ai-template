# 技能：生成计划

## 技能描述
该技能用于根据用户需求生成详细的开发计划，包括任务分解、时间估计和执行顺序。

## 触发指令
- `/plan` - 生成开发计划
- `/todo` - 生成任务列表
- `/estimate` - 估计开发时间

## 输入格式
```
/plan

项目需求：
- 功能1
- 功能2
- 功能3

技术栈：
- 前端：React + TypeScript
- 后端：Node.js + Express
- 数据库：PostgreSQL
```

## 输出格式
```
# 开发计划

## 阶段一：准备工作
1. 环境搭建
2. 项目初始化
3. 技术选型确认

## 阶段二：核心功能开发
1. 功能1实现
2. 功能2实现
3. 功能3实现

## 阶段三：测试与部署
1. 单元测试
2. 集成测试
3. 部署上线

## 时间估计
- 阶段一：1天
- 阶段二：5天
- 阶段三：2天
- 总计：8天
```

## 实现逻辑
1. 分析用户需求
2. 分解任务模块
3. 估计每个任务的时间
4. 生成详细的开发计划
